# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-17

### Added

- Session authentication with active-account validation.
- Responsive administration dashboard and workspace metrics.
- Project and task management with search, filters and pagination.
- Administrator-only team management and activity history.
- Role-based access for administrators, managers and members.
- SQLite demo database, realistic seed data and demo credentials.
- Feature tests and a GitHub Actions workflow.
- Installation, architecture and security documentation.
